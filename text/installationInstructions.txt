1. Through Google Play Store
------------------------------
On Google Play Store tap on the Apps link. Here you can search the app you want to install.
Each app on Google Play Store has its own page where an user can find details, reviews, ratings and download size.
Since GLIMMPSELite is a free app, you will see a button named Install.

While installing the app from web interface, user will be asked to choose a device from drop down menu to install the app on.

Please see the Android documentation at https://play.google.com/store/apps/details?id=com.zeptolab.ctr.paid&feature=app_collections
 for details on Android app installation.

------------------------------
2. Sideloading
------------------------------
User has to go into the device settings and enable “Unknown Sources”.

GLIMMPSELite.apk file will be available for download on SampleSizeShop website at following link http://glimmpse.samplesizeshop.com/
User can then install the downloaded apk file.

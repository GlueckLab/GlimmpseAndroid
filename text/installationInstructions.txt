1. Install from the Google Play Store
------------------------------
Access the Google Play store at 

https://play.google.com/store

Tap on the Apps link and search for GLIMMPSE Lite. Click on the GLIMMPSE Lite icon to 
access the download page.  GLIMMPSELite is a free app, so simply click the install button.

After clicking the install button, select the target device for installation from the drop
down list, and follow the instructions.

For full details on web installation, please see the Android documentation at 
http://support.google.com/googleplay/?hl=en&topic=2803002&rd=1

------------------------------
2. Sideloading
------------------------------
Access the Android device settings and enable “Unknown Sources”.
Download the GLIMMPSELite.apk file from the SampleSizeShop website at 

http://samplesizeshop.org/software-downloads/glimmpse/

Install the downloaded apk file using any of the methods described here

http://www.androidcentral.com/android-apps-install
